CheatsheetTra cứu nhanh

scan & go

Every flag of all four skills, organised by skill. The most-used flags sit in the visible table; the long tail collapses into a disclosure block. Every flag here is verified against its SKILL.md — nothing invented. Need to pick a skill first? Use the skill finder.Mọi cờ của cả bốn skill, xếp theo skill. Các cờ dùng nhiều nhất nằm trong bảng hiện; phần đuôi dài cuộn gọn vào khối disclosure. Mỗi cờ ở đây được kiểm chứng từ SKILL.md của nó — không bịa. Cần chọn skill trước? Dùng bộ chọn skill.

InstallCài đặt

Each skill ships its own installer. All four share one Python venv at .claude/skills/.venv/. Requires Python 3.11+.Mỗi skill đi kèm installer riêng. Cả bốn dùng chung một venv Python tại .claude/skills/.venv/. Cần Python 3.11+.

POSIX (macOS / Linux / WSL)POSIX (macOS / Linux / WSL)
bash .claude/skills/product-spec/install.sh
Windows (PowerShell)Windows (PowerShell)
powershell -ExecutionPolicy Bypass -File .\.claude\skills\product-spec\install.ps1

Replace product-spec with release, product-spec-critique, or telemetry for those skills. Any installer bootstraps the shared venv — re-running is safe (idempotent).Thay product-spec bằng release, product-spec-critique, hoặc telemetry cho các skill đó. Bất kỳ installer nào cũng dựng venv dùng chung — chạy lại an toàn (idempotent).

product-spec flagsCờ product-spec

Invoke with /cleanmatic:product-spec [--flag] [target]. Full guide: product-spec skill page.Gọi bằng /cleanmatic:product-spec [--cờ] [đích]. Hướng dẫn đầy đủ: trang skill product-spec.

FlagCờWhat it doesLàm gì
(no flag)Detect state → present menu (init / BRD / PRD / stories / validate / update / viz / approve / summary).Dò trạng thái → hiện menu (init / BRD / PRD / story / validate / update / viz / approve / tóm tắt).
--productInit or refresh PRODUCT.md (thin product-context labels).Khởi tạo hoặc làm mới PRODUCT.md (nhãn bối cảnh sản phẩm).
--brdCreate or refine the single BRD (business goals + metrics + stakeholders).Tạo hoặc chỉnh sửa BRD duy nhất (mục tiêu kinh doanh + chỉ số + stakeholder).
--prd [feature]Create or refine a PRD for a feature area. Multiple PRDs supported.Tạo hoặc chỉnh sửa PRD cho một vùng tính năng. Hỗ trợ nhiều PRD.
--epic [prd]Create or refine an epic under the given PRD.Tạo hoặc chỉnh sửa một epic dưới PRD đã chỉ định.
--story [epic]Create or refine a story under the given epic.Tạo hoặc chỉnh sửa một story dưới epic đã chỉ định.
--autoBrain-dump → decompose into BRD goals / PRDs / epics / stories; confirm-batch on ambiguous splits.Đổ ý tưởng → phân rã thành mục tiêu BRD / PRD / epic / story; xác nhận theo lô trên các phân chia mơ hồ.
--validateRun structural scripts → layer LLM judgment → human report. Deterministic gate.Chạy script cấu trúc → thêm lớp nhận định LLM → báo cáo người đọc. Cổng tất định.
--approveValidate → open-questions gate → record owner + date → flip status: approved. Nudges --export and --viz afterward.Validate → cổng câu-hỏi-mở → ghi owner + ngày → chuyển status: approved. Nhắc --export và --viz sau đó.
--updateDelta-update: ask what changed → flag affected downstream nodes for PO review. Never auto-rewrites prose.Cập nhật delta: hỏi đã đổi gì → gắn cờ các node xuôi chiều cho PO xem. Không tự ghi lại văn.
--viz <view>Render a visualization (20 views: graph/body/governance/learning). See skill page for the full matrix.Vẽ trực quan hóa (20 khung nhìn: graph/body/quản trị/học). Xem trang skill để biết ma trận đầy đủ.
--summary1-page exec brief. Add --audience release-notes for a delta since last approved.Tóm tắt 1 trang cho lãnh đạo. Thêm --audience release-notes để lấy delta kể từ lần duyệt cuối.
--statusRead-only health nudge: last-validated errors/warns + open-question markers. Does not re-run validate.Kiểm tra sức khỏe chỉ-đọc: lỗi/cảnh báo validate cuối + marker câu-hỏi-mở. Không chạy lại validate.

Invoke with /cleanmatic:product-spec-critique [scope] [--flags]. Full guide: critique skill page.Gọi bằng /cleanmatic:product-spec-critique [phạm vi] [--cờ]. Hướng dẫn đầy đủ: trang skill critique.

FlagCờWhat it doesLàm gì
[scope]Artifact ID to critique (PRD-AUTH, PRD-AUTH-E1, …) or all (default). Full ancestry is always pulled as context.ID artifact để chỉ trích (PRD-AUTH, PRD-AUTH-E1, …) hoặc all (mặc định). Ancestry đầy đủ luôn được kéo làm ngữ cảnh.
--productRun only the product lens (JTBD / Value Prop / Kano / RICE / riskiest-assumption).Chỉ chạy lăng kính product (JTBD / Value Prop / Kano / RICE / giả định rủi ro nhất).
--techRun only the tech lens (INVEST / testability / hidden dependencies / NFR gaps).Chỉ chạy lăng kính tech (INVEST / có thể kiểm thử / phụ thuộc ẩn / khoảng trống NFR).
--marketRun only the market lens (Lean Canvas / Porter-light / Blue Ocean / unit-economics / JTBD-competition).Chỉ chạy lăng kính market (Lean Canvas / Porter nhẹ / Blue Ocean / kinh tế đơn vị / JTBD-cạnh tranh).
--craftRun only the craft lens (clarity / conciseness / consistency / correctness / term drift / vague adjectives).Chỉ chạy lăng kính craft (rõ ràng / súc tích / nhất quán / chính xác / trôi thuật ngữ / tính từ mơ hồ).
--interactiveAskUserQuestion to pick scope + lenses + level before running.AskUserQuestion để chọn phạm vi + lăng kính + mức trước khi chạy.
--level 1..9Voice intensity. Default = 5 (no-mercy). Aliases 1–6: --warm · --gentle · --blunt · --savage · --no-mercy · --roast. Levels 6–9 carry a danger gate (confirm + AskUserQuestion). Level 9 re-confirms on every run. Universal-harm floor applies at all levels.Độ mạnh giọng. Mặc định = 5 (không nhân nhượng). Bí danh 1–6: --warm · --gentle · --blunt · --savage · --no-mercy · --roast. Mức 6–9 có cổng nguy hiểm (xác nhận + AskUserQuestion). Mức 9 hỏi lại mỗi lần chạy. Lằn ranh an toàn phổ quát áp dụng ở mọi mức.
--lang vi|enCritique output language. Default vi. IDs and frontmatter keys stay English.Ngôn ngữ đầu ra critique. Mặc định vi. ID và khóa frontmatter vẫn tiếng Anh.
--no-webDisable the market lens's WebSearch/WebFetch. With no competitors: in BRD, flags missing competitive grounding rather than fabricating.Tắt WebSearch/WebFetch của lăng kính market. Nếu không có competitors: trong BRD, gắn cờ thiếu căn cứ cạnh tranh thay vì bịa.
