# One-tap new note

## User Story

**As a** writer, **I want** to start a new note in one tap, **so that** I never lose a fleeting thought.

## Acceptance Criteria

- Given the app is open, when the writer taps the new-note button, then an empty note opens with the cursor focused in under 200ms.
- Given the writer is typing a note, when they pause or close the app, then the note is auto-saved with no explicit save action.
