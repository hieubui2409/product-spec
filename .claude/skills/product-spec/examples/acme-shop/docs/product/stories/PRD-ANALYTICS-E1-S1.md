# View revenue over time — Story PRD-ANALYTICS-E1-S1

## User Story

**As a** brand-owner, **I want** to see my revenue over time, **so that** I can tell whether the business is growing
and which periods worked.

## Acceptance Criteria

- Given a brand-owner on the dashboard, when they select a date range, then a revenue-over-time chart renders for that range from the rollup tables.
- Given the revenue chart, when the data loads, then the totals reconcile with the brand's payout statements for the same period.
