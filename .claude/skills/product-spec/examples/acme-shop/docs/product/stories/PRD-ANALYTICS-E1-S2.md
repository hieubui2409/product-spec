# Export report CSV — Story PRD-ANALYTICS-E1-S2

## User Story

**As a** brand-owner, **I want** to export my revenue report as a CSV, **so that** I can share it with my accountant or
analyze it in a spreadsheet.

## Acceptance Criteria

- Given a brand-owner viewing a period, when they export the report, then a CSV of revenue, orders, and top products for that period downloads.
- Given the exported CSV, when opened, then its revenue total matches the dashboard figure for the same range.
