# Mobile browse and buy — Story PRD-MOBILE-E1-S1

## User Story

**As a** shopper, **I want** to browse and buy from a followed brand in the app, **so that** I have a fast, native path
to my favorite brands on my phone.

## Acceptance Criteria

- Given a shopper in the native app, when they open a followed brand, then they can browse the catalog in native screens and open product detail.
- Given a shopper ready to buy, when they check out, then the existing web checkout is presented in the app shell and completes the order.
