# Product detail page — Story PRD-CATALOG-E2-S2

## User Story

**As a** shopper, **I want** a clear product page with photos, variants, and stock, **so that** I can decide and add
the right item to my cart.

## Acceptance Criteria

- Given a shopper on a product detail page, when they select an in-stock variant, then the add-to-cart button is enabled and reflects the variant price.
- Given a product with multiple photos, when the page loads on mobile, then the gallery is swipeable and images lazy-load below the fold.
