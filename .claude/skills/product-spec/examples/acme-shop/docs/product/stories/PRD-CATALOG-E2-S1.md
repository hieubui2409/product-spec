# Browse the catalog — Story PRD-CATALOG-E2-S1

## User Story

**As a** shopper, **I want** to browse and search a brand's catalog, **so that** I can find the piece I want quickly.

## Acceptance Criteria

- Given a shopper on a brand storefront, when they browse a category, then published products in that category are listed with photo, name, and price.
- Given a shopper using in-catalog search, when they enter a query, then matching published products are returned within 1 second.
